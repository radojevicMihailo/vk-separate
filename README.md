# vk-separate

1. Clone the repo

```sh
git clone https://github.com/radojevicMihailo/vk-separate

```

2. Add vk.json into the root of the `vk-separate`

3. Run command

```sh
cargo run <vk_json_file_route> <output_route_files> <output_file_name>
# e.g.
# cargo run vk.json ./ vk_without_cks
```
